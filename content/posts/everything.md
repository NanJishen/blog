## 设置

- 常规 - 随系统启动；Everything 服务
- 界面 - 单击托盘图标打开搜索窗口
- 结果 - 搜索关键词为空时不显示搜索结果；双击路径列打开目录
- 视图 - 高亮光标经过行；大小格式“自动”
- 字体与颜色 - 项目状态 
	- 高亮部分：前景色红色，背景色淡黄色
	- 鼠标悬停：背景色 `210，233，255`
- 历史 - 启用搜索历史 10 天；关闭运行历史
- 索引：数据库路径：`T:\cache`；
- 索引 - 文件夹 - 添加，如 NAS 目录
- 索引 - 排除列表 - 排除不需要搜索的“文件夹”（如 D:\Games）和“筛选器”（`*.lnk`）

## 搜索语法示例

| 语法 / 符号 | 说明        | 示例                                     |
| ------- | --------- | -------------------------------------- |
| `*`     | 通配符（任意字符） | `*.pdf`（所有 PDF 文件）                     |
| `空格`    | 与（AND）    | `01 ok`（同时包含 01 和 ok）                  |
| `\|`    | 或（OR）     | `123\|456`（包含 123 或 456）               |
| `dm:`   | 日期范围      | `*.jpg dm:2016/10/10-:2016/10/20`      |
| `X:\XX` | 指定路径      | `D:\folder 123`（D:\folder 下包含 123 的文件） |
| `size:` | 文件大小      | `*.zip size:>300MB`                    |
| `?`     | 单个字符通配符   | `???.txt`（3 个字符的 .txt 文件）              |
| `.`     | 任意单个字符    | `nj.hen`（如 njshen）                     |
| `^`     | 开头匹配      | `^game`（以 game 开头）                     |
| `$`     | 结尾匹配      | `game$`（以 game 结尾）                     |
| `[ ]`   | 包含字符      | `[abc]`（包含 a / b / c）                  |
| `[^ ]`  | 排除字符      | `[^abc]`（不包含 a / b / c）                |
| `[-]`   | 范围        | `[a-z]`（小写 a–z）                        |
| `[--]`  | 多范围       | `[a-zA-Z]`（大小写字母）                      |
