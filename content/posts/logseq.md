## Journals

左侧面板的  Journals 为日记分类。如果你不需要可以在设置中关闭。它会每天自动生成对应日期的页面。你可以将每天所想所遇随手记录，聚沙成塔。

## All pages

左侧面板的 All pages 点击后可以看到所有页面：
- Contents ：目录页，编辑该页就是编辑右侧面板的目录
- favorites ：喜欢页，该页（待补充）

关于页面的使用：
- 可以先建立块 `[[新页面名]]`，然后单击该块就会新建对应的新页面
- 在页面中点右上角的 `···` ，接着点击删除即可删除该页
- 页面名尽量不要使用特殊符号，因为页面名也是文件名，不支持一些特殊符号，例如 `.`

## 块概念

这是双链笔记的主要用法。将重要的“关键字”用双方括号括起来 `[[关键字]]` 。日后点击该块即可跳转到对应页面。添加块的方法：
- 无论中英文下按连按两次 `[[ 或 【【 ` 即可添加块
- 框选你要设置块的关键字，按 `Alt + [[ 或 【【`

块引用：可用 `(( ))` 将要被引用的内容括起来。在被引用的块的右上角，会显示当前块被引用了几次。单击数字，会打开右侧边栏。

块内换行可以用 `Shift+Enter`

右键左侧小点可以弹出右键菜单：
- 可以设置条目的颜色
- 可以拷贝引用，然后用 Ctrl+V 粘贴即可
- 可以右键块在右侧边栏打开
- 右键点击条目左侧的点可以将当前保存为模板，方便日后直接调用。因此你应该先设置好作为模板的样式和内容在去转为模板。其它页面调用模板只需 `/Template` 即可。

## 命令语法

输入左斜杠 `/` 可以调用命令，其中包含了很多，例如插入图片，日期等：
- `/PAG` 会筛选出页面链接语法，有“页面内容引用”和“页面链接引用”可选
- `/BL` 会筛选出块的链接语法，有“块内容引用”和“块链接引用”可选

输入 `<` 尖括号可以调用出另一类命令，能够插入例如 小 Note，Tips 等小注释引用。

以上只是简单的列举，你可以自行看看。有很多的命令，这样调用起来会非常方便。

## 设置 TODO

先输入内容，然后 `/todo` 即可将该内容设置为 TODO 条目。当然，也可以先输入 `/todo` 再输入内容。还可以设置 `/later` `/now` `/done` 等（更多状态可以在 / 命令菜单中查到）。然后可以通过点击标记完成。

设置优先级的：光标在 TODO 条目结尾处，输入 `/C` （有 A、B、C 三个级别，可以通过鼠标悬停上方，点选切换。

设置截止日期：光标在 TODO 条目结尾处，输入 `/DEADLINE` 。

TODO 条目可能分散于各处。可以通过 Ctrl+K 键打开全局搜索 todo 关键字。接着你会看到结果中最上面的一条左面带一个 P 符号（表示页面），点击进入该页后可以看到所有 TODO 都会汇总在这里。

## 查询内容

输入 `/query` 可以查询内容，并插入到页面中。例如 `{{query todo}}`，就可以查询 todo 条目。你可以在其它页面插入该查询，将鼠标指向查询可以看到更多用法：
```bash
{{query #tag}}
{{query [[page]]}}
{{query "full-text search"}}
{{query (and [[project]] (task NOW LATER))}}
{{query (or [[page 1]] [[page 2]])}}
{{query (and (between -7d +7d) (task DONE))}}
{{query (property key value)}}
{{query (page-tags #tag)}}
```

## 收藏页面

右上角点击 ... ，然后点“添加收藏”，可以将当前页面添加到左侧面板中的“收藏页面”。例如建立一个页面叫做 Dashboard 做为仪表盘。然后在上面添加一些查询内容，比如 `{{query todo}}` 。

## 标签与分类

打标签：先输入一个 `#` 号，然后紧挨着输入标签内容，例如 `#标签` 。你也可以将标签页添加到收藏页面，方便快速点击跳转

分类：
```bash
type:: 心理学, 幸福课 # 可以英文逗号分割多个类似
author:: 泰勒·本·沙哈尔
publish:: 中信出版社
mark:: 书籍
```

那么如何查询呢？可以参考[[#查询内容]]段落。

## 设置卡片

首先输入一个问题，例如：`hello 是什么意思？` ，然后在后面输入 `#card`，即可将该问题设置为卡片。接着回车换行，再按`TAB`键缩进，然后输入答案，最终格式：
```bash
hello 是什么意思？ #card
	它的英文是：您好
```

不仅如此，还可以给答案增加一个填空，先按照上述方法建立卡片，在答案中可以输入`/cloze`然后输入填空中的答案最终格式格式为：
```bash
一些问题 #card 
	{{cloze 填空中的答案}} 以及 {{cloze 填空中的答案}}
```

有的卡片和答案后，点击左侧导航面板中的 Flashcards 即可看到这些设置的卡片。

## 基本编辑

编辑器是支持 Emoji 表情的，所以你可以尽情使用。

插入图片：先复制图片，然后使用 Ctrl+V 直接粘贴到编辑器中。图片也会被同时复制到所在库的根目录中的 assets 目录中。

插入表格：
```bash
|*标题*|*标题*|*标题*|
|内容|内容|内容|
|内容|内容|内容|
|内容|内容|内容|
```

插入文件：可以直接将文件拖入到编辑器中，同样会被复制到所在库的根目录中的 assets 目录中。

## 引用 pdf 文件

直接将 PDF 文件拖入编辑器中，之后鼠标悬停即可直接查看文件。点击该文件可以进入阅读模式。不仅如此，你还可以框选文字标注颜色，然后回到编辑器里 `Ctrl + V`，引用这个文件的段落，那么当点击段落后，即可跳转到该段落。

你也可以输入以插入文件：`file:: [xxx.pdf](../assets/xxx.pdf)`，以这样的形式插入，点击后会用浏览器直接打开 PDF 文件。

## 快捷键

在设置 - 常规 - 快捷键设置，可以查看和自定义快捷键

| 按键 | 说明 |
| --- | --- |
| Ctrl+, | 打开设置 |
| Ctrl+C，Ctrl+B | 切换编辑器是否显示 `[[ ]]` 块括号 |
| Ctrl+F | 搜索当前页面并高亮关键字 |
| Ctrl+K | 打开全局搜索/创建页面 |
| Ctrl+Shift+K | 搜索当前页面搜索 |
| Ctrl+Shift+P | 命令面板 |
| Shift+鼠标左键点击`[[ ]]`块 | 右侧新窗口打开该块页面 |
| Ctrl+↑ 和 Ctrl+↓ | 折叠 和 展开 |
| Ctrl+Shift+= 或 Ctrl+- | 放大或缩小界面，Ctrl+0 恢复默认大小 |
| Ctrl+N | 打开一个新的 Logseq 实例 |

## 插件与主题

右上角 .... - 插件，可以查看到已安装的插件和主题，及访问插件和主题市场。
右上角 .... - 主题，可以切换已安装的主题。

插件推荐：
[Markdown Table Editor](https://github.com/haydenull/logseq-plugin-markdown-table)：编辑表格更直观
[Bullet Threading](https://github.com/pengx17/logseq-plugin-bullet-threading) ：列表添加指向线

主题推荐：
[Bonofix theme](https://github.com/sansui233/logseq-bonofix-theme)：简洁专注，适用于子弹笔记和写作
[Dev theme](https://github.com/pengx17/logseq-dev-theme)：简洁专注，在 MAC 上有更好的体验
[Woz theme](https://github.com/haydenull/logseq-woz-theme)：基于 Dev Theme 的简洁，但阅读更舒适
[Cobra](https://github.com/santiyounger/Cobra)：基于 Obsidian 的 [Wasp]( https://github.com/santiyounger/Wasp-Obsidian-Theme) 主题
[Monokai](https://github.com/tzcl/logseq-monokai)：暗黑主题，受到 Monokai 和 Dracula 的启发
